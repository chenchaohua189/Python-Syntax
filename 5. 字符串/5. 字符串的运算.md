### 1. 字符串的加法可以将两个字符串拼接起来，得到一个新字符串。
```python
print("hello " + "world")   # hello world

```

### 2. 字符串乘以一个整数，可以将若干个自身拼接起来，得到一个新字符串。
```python
_str = "python"

print(_str * 2) # pythonpython

```

### 3. 字符串支持比较运算符，按字典序比较大小。即如果两个字符串相同，则表示相等；否则找到两个字符串从左到右数第一个不一样的字符，哪个字符串的字符的ASCII码小，哪个字符串的字典序就小；另外空字符比任何字符都小。
```python
print("python" > "acwing")  # True

```
