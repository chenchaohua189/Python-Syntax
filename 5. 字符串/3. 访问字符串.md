### 1. 通过下标(-n, n - 1)访问(当下标元素为负数时, 就是对字符串长度取模的结果)
```python
_str = "python"

print(_str[0])  # p
print(_str[-2]) # o
```

### 2. 使用for...in...循环遍历
#### (1) 使用range()函数
```python
_str = "python"

for i in range(len(_str)):
    print(_str[i], end=" ")  # p y t h o n
```

#### (2) 使用for...in...直接遍历字符串
```python
_str = "python"

for ch in _str:
    print(ch, end=" ")  # p y t h o n
```
