### `a[begin, end, step]`(`begin`: 起始下标(包含), `end`:结束下标(不包含), `step`: 步长(正数代表正方向, 负数代表反方向))
- 省略begin时, begin的默认值是0
- 省略end时, end的默认值是列表长度
- 如果beign或end是负数, 是对列表长度的取模操作

### 1.
```python
_str = "python"

print(_str[0:6]) # python
print(_str[:6]) # python
print(_str[0:]) # python
print(_str[:]) # python
print(_str[-6:0]) # python
```
