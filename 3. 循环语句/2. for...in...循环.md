## for...in... 循环: 遍历可迭代对象
### 1. 遍历字符串
```python
for ch in "python":
    print(ch, end=" ")  # p y t h o n
```

### 2. 遍历列表
```python
for x in [1, 2, 3]:
    print(x, end=" ")  # 1 2 3 
```

### 3. 遍历range(x, y, z) 返回的可迭代对象
#### (1) range(x): 返回 0, 1, ..., x - 1 数列
```python
for i in range(5):
    print(i, end=" ")  # 0 1 2 3 4
```

#### (2) range(x, y): 返回 x + 1, x + 2, ..., y - 1 数列
```python
for i in range(1, 6):
    print(i, end=" ")  # 1 2 3 4 5
```

#### (3) range(x, y, z)
- z > 0: 返回x, x + z, x + 2z, ... 小于y的数列

```python
for i in range(0, 10, 2):
    print(i, end=" ")  # 0 2 4 6 8
```

- z < 0: 返回x, x + z, x + 2z, ... 大于y的数列

```python
for i in range(10, -1, -2):
    print(i, end=" ")  # 10 8 6 4 2 0
```
