### `a[begin, end, step]`(`begin`: 起始下标(包含), `end`:结束下标(不包含), `step`: 步长(正数代表正方向, 负数代表反方向))
- 省略begin时, begin的默认值是0
- 省略end时, end的默认值是元组长度
- 如果beign或end是负数, 是对元组长度的取模操作

## 1. 切片读
```python
a = (1, 2, 3)

print(a[:len(a)])   # (1, 2, 3)
print(a[0:])    # (1, 2, 3)
print(a[-1:-2:-1]) # (3, )
```
