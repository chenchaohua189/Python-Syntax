### 1. 元组 + 元组: 将两个元组拼接起来, 得到一个新元组
```python
a = (1, 2, 3)
b = a + a

print(b)    # (1, 2, 3, 1, 2, 3)
```

### 2. 元组 * 整数: 将若干个自身拼接起来, 得到一个新元组
```python
a = (1, 2, 3)

print(a * 2)    # (1, 2, 3, 1, 2, 3)
```

### 3. 元组的比较运算: 逐个元素比较
```python
a = (1, 2, 3)
b = (4, 5, 6)

print(a < b)    # True
```
