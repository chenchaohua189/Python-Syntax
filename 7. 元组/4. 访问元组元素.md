### 1. 通过下标-n ~ n - 1访问(同列表)
```python
a = (1, 2, 3)

print(a[0]) # 1
print(a[-1])    # 3
```

### 2. 通过for...in...访问元组元素(同列表)
- (1) 使用range()函数通过下标遍历元组
```python
a = (1, 2, 3)

for i in range(0, len(a)):
    print(a[i], end = " ")  # 1 2 3
```

- (2) 使用for...in...遍历直接遍历元组
```python
a = (1, 2, 3)

for x in a:
    print(x, end = " ") # 1 2 3 
```
