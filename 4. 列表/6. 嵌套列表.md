## 1. 嵌套列表的初始化
### (1) 直接初始化
```python
a = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

print(a)    # [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
```

### (2) 使用for...in初始化
```python
a = [[(i * 3 + j + 1) for j in range(3)] for i in range(3)]

print(a)    # [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
```
