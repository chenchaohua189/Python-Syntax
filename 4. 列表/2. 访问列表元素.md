### 1. 通过下标(-n ~ n - 1)访问单个列表元素(当下标为负数的时候, 就是下标对列表长度的取模运算)
```python
a = [1, 2, 3]

print(a[0]) # 1
print(a[-2]) # 2

```

### 2. 通过 for...in 循环遍历列表元素
- (1) 使用range()函数通过下标遍历列表
```python
a = [1, 2, 3]

for i in range(len(a)):
    print(a[i], end=" ")  # 1 2 3
```

- (2) 使用for...in遍历直接遍历列表
```python
a = [1, 2, 3]

for x in a:
    print(x, end=" ")  # 1 2 3
```
