### `a[begin, end, step]`(`begin`: 起始下标(包含), `end`:结束下标(不包含), `step`: 步长(正数代表正方向, 负数代表反方向))
- 省略begin时, begin的默认值是0
- 省略end时, end的默认值是列表长度
- 如果beign或end是负数, 是对列表长度的取模操作

## 1. 切片读(返回新列表)
```python
a = [1, 2, 3, 4, 5]

print(a[:2])    # [1, 2]
print(a[3:])    # [4, 5]
print(a[:]) # [1, 2, 3, 4, 5]
print(a[1::2]) # [2, 4]
print(a[::-1])  # [5, 4, 3, 2, 1]
```

## 2. 切片写(修改原列表)
```python
a = [1, 2, 3, 4, 5]

a[2:] = [7, 8]  

print(a)    # [1, 2, 7, 8]
```
