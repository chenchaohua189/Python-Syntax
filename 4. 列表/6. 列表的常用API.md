(1) len(a): 返回列表长度
```python
a = [1, 2, 3]

print(len(a))   # 3
```

(2) append(): 在列表末尾添加一个元素
```python
a = [1, 2, 3]

a.append(4)

print(a)   # [1, 2, 3, 4]
```

(3) pop(): 删除列表的最后一个元素
```python
a = [1, 2, 3, 4]

a.pop()

print(a)    # [1, 2, 3]

```

(4) a.reverse(): 反转列表
```python
a = [1, 2, 3]

a.reverse()

print(a)    # [3, 2, 1]

```

(5) a.sort(): 将列表从小到大排序
```python
a = [2, 3, 1]

a.sort()

print(a)    # [1, 2, 3]

```
