### 1. 位置实参初始化形参
```python
def f(a, b):
    print(f"a = {a}, b = {b}")

f(1, 2) # a = 1, b = 2

```

### 2. 变量名实参初始化形参
```python
def f(a, b):
    print(f"a = {a}, b = {b}")

f(a = 1, b = 2)  # a = 1, b = 2
```

### 3. 位置实参和变量名实参混用初始化形参(此时位置实参需要在关键字实参之前)
```python
def f(a, b):
    print(f"a = {a}, b = {b}")

f(1, b = 2)  # a = 1, b = 2
```

### 4. 默认值初始化形参
```python
def f(a = 1, b = 2):
    print(f"a = {a}, b = {b}")

f()    # a = 1, b = 2

```
