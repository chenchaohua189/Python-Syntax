### 算术运算(支持赋值运算)
| 运算符 | 描述 | 
| - | - |
| `+` | 加 |
| `-` | 减 | 
| `*` | 乘 | 
| `/` | 除(结果为float) |
| `//` | 整数(结果为int) | 
| `%` | 取余(运算结果的符号与除数一致) | 
| `**` | 乘方 | 

```python
a = 10
b = 20

print(a + b)    # 30
print(a - b)    # -10
print(a * b)    # 200
print(a / b)    # 0.5
print(a // b)   # 0
print(a % b)    # 10
print(a ** b)   # 1e20
```
