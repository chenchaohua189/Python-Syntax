### 比较运算(支持链式比较)

| 运算符 | 描述 | 
| - | - |
| `>` | 大于 | 
| `>=` | 大于等于 | 
| `<` | 小于 | 
| `<=` | 小于等于 | 
| `==` | 等于 | 
| `!=` | 不等于 | 

```python
a = 10
b = 20

print(a > b)    # False
print(a >= b)   # False
print(a < b)    # True
print(a <= b)   # True
print(a == b)   # False
print(a != b)   # True
```
