## 1. print(a, b, sep, end)函数: sep自定义分隔符(默认空格), end自定义行末符号(默认换行)
### (1) print()函数的使用
```python
a = 10
b = 20

print(a, b) # 10 20
```

### (2) print()函数自定义分隔符
```python
a = 10
b = 20

print(a, b, sep=", ") # 10, 20
```

### (3) print()函数自定义行末符号
```python
a = 10
b = 20

print(a, b, end="!") # 10 20!
```

## 2. f-string格式化输出
```python
a = 10
b = 20

print(f"a = {a}, b = {b}")  # a = 10, b = 20
```
